# Product Requirements Document

## Product
Security Posture Tool

## Goal
Provide a lightweight assessment experience that helps organizations understand their current cyber maturity and priority improvement areas.

## MVP Scope
- Landing page
- Company capture form
- 12-question assessment
- Weighted scoring model
- Results page with recommendations
- Leaderboard

## Future Scope
- PDF reports
- Shareable badge
- Invite flow
- Multi-tenant accounts
- Historical trends
