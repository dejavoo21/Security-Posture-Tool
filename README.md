# Security Posture Tool

Starter full-stack app for a lightweight security posture assessment platform.

## Stack
- Frontend: React + Vite + TypeScript
- Backend: Node.js + Express + TypeScript

## Run locally

### Backend
```bash
cd backend
npm install
npm run dev
```
Backend runs on `http://localhost:4000`.

### Frontend
```bash
cd frontend
npm install
npm run dev
```
Frontend runs on `http://localhost:5173`.

## Features included
- Landing page
- Start assessment form
- Question flow
- Weighted scoring
- Results page with domain breakdown
- Action recommendations
- In-memory sample data
